# 🧠 Online Quiz Platform  
A sleek, responsive web application for taking interactive quizzes on various topics.  
# ✨ Features  
## Dynamic Quiz Generation:  
Loads questions dynamically.
## User Authentication:  
Secure login for users to save progress.
## Real-time Feedback:  
Immediate feedback on answers (correct/incorrect).
## Result Dashboard:  
Displays score, correct answers, and final grade.
## Fully Responsive:  
Optimized for both mobile and desktop views.

# 🚀 Installation & Setup  
Follow these steps to run the project locally.
## Clone the repository:  
bash
git clone https://github.com/your-username/your-quiz-repo.git
## Navigate to project folder:  
bash
cd your-quiz-repo
## Install dependencies:  
bash
npm install
## Setup Environment Variables:  
Create a .env file and add your API keys/Database URI:
env
DB_URI=your_mongodb_uri
API_KEY=your_api_key
Run the application:
bash
npm start
 
# 🎮 How to Use  
Open the application in XAMPP server  
Select a quiz category.  
Answer the questions.  
View your final results and score.
